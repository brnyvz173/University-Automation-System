# 🎓 University Automation System

A SQL Server-based University Automation System developed to practice relational database design, SQL queries, views, stored procedures, and triggers.

This project simulates a university management system where students, instructors, departments, courses, and enrollments are managed through a relational database.

---

# 📌 Project Features

- Create and manage university database
- Department management
- Student management
- Instructor management
- Course management
- Course enrollment management
- SQL queries for reporting
- SQL Views
- Stored Procedures
- Database Triggers

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/brnyvz173/University-Automation-System.git
```

Open SQL Server or Azure Data Studio.

Run the SQL files in the following order:

1. create_tables.sql
2. insert_data.sql
3. queries.sql
4. views.sql
5. procedures.sql
6. triggers.sql
